Skin Deep source code
Blendo Games
Feb 13, 2025

This is the source code to Skin Deep (2025), a first-person immersive sim game. The game will be available for purchase at Steam.

Skin Deep uses idTech4, otherwise known as the Doom 3 engine. It uses code from Dhewm3 and The Dark Mod.
